# ATM Management System in C

A simple menu-driven ATM Management System developed using the C programming language.

---

## Features

- Check Account Balance
- Deposit Money
- Withdraw Money
- Prevents overdrawing
- Menu-driven interface
- Uses Functions
- Uses Switch Case
- Uses Do-While Loop

---

## Technologies Used

- C Programming Language
- Cxxdroid (Android)

---

## How to Run

1. Compile the program.
2. Run the executable.
3. Choose an option from the menu.
4. Perform ATM operations.
5. Exit the program.

---

## Sample Menu

```
==============================
      ATM MANAGEMENT SYSTEM
==============================
1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit
```
